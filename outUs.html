Блог о нас и немного о PHP

Поговорим о PHP

В этой статье вы узнаете:

  • Что такое PHP, где его используют
  • Для чего создан PHP
  • В каких программах на нём пишут

    • Итак, PHP - Скриптовый язык общего назначения, интенсивно применяемый для разработки веб-приложений. Поддерживается большинством хостинг-провайдеров и является одним из лидеров среди языков, применяющихся для создания динамических веб-сайтов. Данный язык используется для серверной части сайта, может как внедряться непосредственно в html, так и работать отдельным файлом. С помощью PHP, при разраюотке сайта, можно работать с такими функциями как:

  • работа с авторизацией
  • работа с cookies и сессиями
  • обработка файлов, загружаемых на сервер
  • взаимодействиями с различными системами управления базами данных через доп. модули

    • И многими другими...
